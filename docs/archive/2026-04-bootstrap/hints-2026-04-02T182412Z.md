## Important Observations

- XML-RPC login works with password format `$1$` plus MD5 hex of the raw password
- `EventQueueGet` empty polls currently return `502 BadGateway` from local OpenSim, which should be treated as normal empty-state behavior
- the first successful UDP handshake burst already returns:
  - `PacketAck`
  - `RegionHandshake`
  - `AgentMovementComplete`
  - `ParcelOverlay`
  - `ObjectUpdate`
- `UseCircuitCode` alone gets an ACK, but `CompleteAgentMovement` is what moves the session into useful simulator replies
